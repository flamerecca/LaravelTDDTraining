# Laravel TDD 教材

## 準備步驟

`git clone` 這個專案之後，運行

`
$ composer update
`

之後運行

`
$ vendor/phpunit/phpunit/phpunit
`

應該可以看到

```
PHPUnit 7.5.15 by Sebastian Bergmann and contributors.

..                                                                  2 / 2 (100%)

Time: 160 ms, Memory: 16.00 MB

OK (2 tests, 2 assertions)
```
